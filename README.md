# 📂 10-Minute Delivery Shopping App

This repository contains an **online shopping application** based on a 10-minute delivery concept, inspired by instant grocery delivery platforms. The app focuses on providing a quick shopping experience with fast product selection and cart management.

---

## 📚 About

The purpose of this project is to practice building an instant delivery style e-commerce application, where users can browse products and place orders designed for rapid delivery services.

---

## 🚀 Features

* Instant product browsing
* Category-wise product listing
* Product details view
* Add to cart functionality
* Go to cart option
* Quick shopping UI experience

---

## ⚡ Delivery Concept

This app is designed around a **10-minute delivery model**, simulating how instant grocery delivery platforms operate.

---

## 🛠️ Tech Used

* Java
* XML
* Android Studio
* Firebase / Sqlite 

---

## 🎯 Learning Outcome

Through this project, I learned:

* Instant delivery app structure
* Shopping cart flow
* Product data handling
* UI design for quick-commerce apps
